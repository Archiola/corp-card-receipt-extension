자동 카드 매칭

등록한 카드로 결제한 주문만 선별합니다. BC카드·비씨카드 등 동의어도 자동으로 처리합니다.

⚡

클릭 한 번

영수증 창 열기·캡처·다운로드까지 모든 과정이 백그라운드에서 자동으로 진행됩니다.

📁

체계적 파일 정리

쇼핑몰·카드별 폴더로 자동 분류하고 날짜·금액·상품명으로 파일명을 만듭니다.

💳

멀티카드 지원

여러 장의 카드를 등록하고 동시에 처리할 수 있습니다.

사용 방법

4단계면 충분합니다.

STEP 01

카드는 여러 장 등록할 수 있습니다.
별칭은 저장 폴더명에 사용됩니다. (예: 홍길동 카드)
BC카드·비씨카드처럼 같은 카드의 다른 표기는 자동 처리됩니다.

설치 방법

3분이면 됩니다.

1

파일 준비

확장프로그램 폴더를 원하는 위치에 저장합니다.

2

Chrome 설정 열기

주소창에 chrome://extensions 를 입력하고 "개발자 모드"를 켭니다.

3

폴더 불러오기

"압축해제된 확장 프로그램을 로드합니다"를 클릭하고 폴더를 선택합니다.

4

아이콘 고정

Chrome 도구 모음의 퍼즐 아이콘을 클릭하고 고정 아이콘(📌)을 눌러 고정합니다.

지원 쇼핑몰

두 곳을 지원합니다.

🛒 쿠팡

기업 전용 쿠팡(mc.coupang.com)의 주문 목록에서 작동합니다. 주문 상세 페이지에서 카드사를 확인하고 거래명세서를 자동 캡처합니다.

mc.coupang.com 거래명세서 카드사 검증

🛍 네이버쇼핑

네이버쇼핑 주문 내역에서 카드영수증을 자동으로 열고 캡처합니다. 카드사와 뒷 4자리를 모두 검증한 뒤 저장합니다.

shopping.naver.com 카드영수증 뒷번호 검증
