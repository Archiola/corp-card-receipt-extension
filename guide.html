🎯

자동 카드 매칭

등록한 카드로 결제한 주문만 선별합니다. BC카드·비씨카드 등 동의어도 자동으로 처리합니다.

⚡

클릭 한 번

영수증 창 열기·캡처·다운로드까지 모든 과정이 백그라운드에서 자동으로 진행됩니다.

📁

체계적 파일 정리

쇼핑몰·카드별 폴더로 자동 분류하고 날짜·금액·상품명으로 파일명을 만듭니다.

💳

멀티카드 지원

여러 장의 카드를 등록하고 동시에 처리할 수 있습니다.

사용 방법

4단계면 충분합니다.

STEP 01

카드를
등록합니다.

확장프로그램 아이콘을 클릭해 팝업을 열고, 카드사·뒷 4자리·별칭을 입력합니다.

카드는 여러 장 등록할 수 있습니다.
별칭은 저장 폴더명에 사용됩니다. (예: 홍길동 카드)
BC카드·비씨카드처럼 같은 카드의 다른 표기는 자동 처리됩니다.

설치 방법

처음 설치해도
어렵지 않습니다.

1

확장프로그램 폴더를 받습니다

담당자에게서 받은 폴더(또는 압축 파일)를 컴퓨터의 원하는 위치에 저장합니다.
압축 파일이라면 먼저 압축을 풀어 폴더 형태로 만들어 주세요.

⚠️ 나중에 폴더를 옮기면 확장프로그램이 오류가 날 수 있습니다. 최종 위치에 저장하세요.

2

Chrome 확장프로그램 관리 페이지를 엽니다

Chrome 브라우저를 열고 주소창을 클릭한 뒤, 아래 주소를 그대로 복사하여 붙여넣고 Enter를 누릅니다.

chrome://extensions
⚠️ 검색창이 아닌 주소창에 입력해야 합니다.
chrome://extensions
Enter 키를 누르세요 →
3

개발자 모드를 켭니다

확장프로그램 관리 페이지 오른쪽 위에 있는 "개발자 모드" 토글을 클릭하여 켭니다.
켜지면 토글이 파란색으로 바뀌고 새로운 버튼 3개가 나타납니다.

확장 프로그램
개발자 모드
압축해제된 확장 프로그램을 로드합니다
4

"압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다

버튼을 클릭하면 폴더 선택 창이 열립니다.
1단계에서 저장해 둔 확장프로그램 폴더를 찾아 선택하고 "폴더 선택" 버튼을 클릭합니다.

⚠️ 폴더 안의 파일이 아닌 폴더 자체를 선택해야 합니다.

카드 오른쪽 하단의 토글이 파란색인지 확인하세요. 회색이면 클릭하여 켭니다.
